# Sound Files

Place your sound files in this directory:

- `betting.mp3` - Background sound for betting screen
- `race.mp3` - Background sound for race screen (horse galloping, crowd cheering)
- `win.mp3` - Victory sound when player wins
- `lose.mp3` - Loss sound when player loses
- `summary.mp3` - Background sound for summary screen

## Format
- Supported formats: MP3, WAV
- Recommended: MP3 format for smaller file size

## Sources
You can find free sound effects at:
- https://freesound.org
- https://pixabay.com/music/
- https://mixkit.co/free-sound-effects/

## Note
If sound files are not provided, the game will continue to work but without sound effects.
